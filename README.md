# 🖥️ Pomodoro de Estudos

## 📜 Descrição
Projeto realizado com react e typescript, onde o usuário pode listar o que precisa estudar e pode criar um time com cronômetro regressivo.

## ✍️ Processo
- React
- TypeScript
- Sass
- Css modules
- Design responsivo

### 🎲 Funcionalidades
 - Formulário
 - Lista
 - Cronômetro
